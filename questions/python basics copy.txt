Q: What is Python?
A: Python is a high-level, interpreted, object-oriented programming language known for its simplicity, readability, and extensive standard library.

Q: Who developed Python?
A: Python was developed by Guido van Rossum and first released in 1991.

Q: What are the main features of Python?
A: Python is easy to learn, interpreted, object-oriented, platform-independent, dynamically typed, open-source, and supports a large standard library.

Q: What is the difference between compiled and interpreted languages?
A: Compiled languages are converted into machine code before execution, while interpreted languages execute code line by line. Python is an interpreted language.

Q: Why is Python called a platform-independent language?
A: Python programs can run on different operating systems such as Windows, Linux, and macOS without modifying the source code.

Q: What are variables in Python?
A: Variables are names used to store data values. Python automatically determines the variable's data type based on the assigned value.

Q: What are the basic data types in Python?
A: Common data types are int, float, str, bool, list, tuple, set, dictionary, and NoneType.

Q: What is dynamic typing in Python?
A: Dynamic typing means you do not need to declare the data type of a variable. Python determines the type at runtime.

Q: What is the difference between int and float?
A: int stores whole numbers, while float stores numbers with decimal points.

Q: What is a string in Python?
A: A string is a sequence of characters enclosed in single quotes, double quotes, or triple quotes.

Q: What is a list in Python?
A: A list is an ordered, mutable collection that allows duplicate elements.

Q: What is a tuple in Python?
A: A tuple is an ordered, immutable collection that allows duplicate elements.

Q: What is a set in Python?
A: A set is an unordered collection of unique elements. Duplicate values are automatically removed.

Q: What is a dictionary in Python?
A: A dictionary is a collection of key-value pairs. Keys are unique, and values can be of any data type.

Q: What is the difference between a list and a tuple?
A: Lists are mutable and can be modified after creation, whereas tuples are immutable and cannot be changed.

Q: What are Python keywords?
A: Keywords are reserved words with predefined meanings, such as if, else, for, while, def, class, and return.

Q: What is indentation in Python?
A: Indentation refers to the spaces at the beginning of a code block. It defines the structure of the program and is mandatory in Python.

Q: What are comments in Python?
A: Comments are non-executable lines used to explain code. Single-line comments begin with #, while multi-line comments use triple quotes.

Q: What is the input() function?
A: The input() function reads input from the user as a string.

Q: What is the print() function?
A: The print() function displays output on the screen.

Q: What are operators in Python?
A: Operators are symbols used to perform operations on variables and values, such as arithmetic, comparison, logical, assignment, and bitwise operations.

Q: What are arithmetic operators in Python?
A: Arithmetic operators include +, -, *, /, %, //, and ** for performing mathematical calculations.

Q: What are comparison operators in Python?
A: Comparison operators compare two values and return True or False. They include ==, !=, >, <, >=, and <=.

Q: What are logical operators in Python?
A: Logical operators are and, or, and not. They are used to combine or negate conditional expressions.

Q: What is type casting in Python?
A: Type casting is the process of converting one data type into another using functions such as int(), float(), str(), and bool().

Q: What are conditional statements in Python?
A: Conditional statements are used to make decisions in a program. The main conditional statements are if, elif, and else.

Q: What is an if statement?
A: The if statement executes a block of code only if the specified condition is True.

Q: What is the purpose of the elif statement?
A: The elif statement checks additional conditions if the previous if or elif condition is False.

Q: What is the purpose of the else statement?
A: The else statement executes a block of code when all previous conditions are False.

Q: What is a nested if statement?
A: A nested if statement is an if statement placed inside another if statement to check multiple conditions.

Q: What is a loop in Python?
A: A loop repeatedly executes a block of code until a specified condition is met.

Q: What is a for loop?
A: A for loop is used to iterate over a sequence such as a list, tuple, string, or range.

Q: What is a while loop?
A: A while loop repeatedly executes a block of code as long as the specified condition is True.

Q: What is the range() function?
A: The range() function generates a sequence of numbers and is commonly used with for loops.

Q: What is the difference between a for loop and a while loop?
A: A for loop is used when the number of iterations is known, while a while loop is used when the number of iterations depends on a condition.

Q: What is the break statement?
A: The break statement immediately terminates the nearest loop and transfers control to the next statement after the loop.

Q: What is the continue statement?
A: The continue statement skips the current iteration of a loop and continues with the next iteration.

Q: What is the pass statement?
A: The pass statement is a null statement that does nothing. It is used as a placeholder where code is required syntactically.

Q: What is a function in Python?
A: A function is a reusable block of code that performs a specific task. It is defined using the def keyword.

Q: How do you define a function in Python?
A: A function is defined using the def keyword followed by the function name, parentheses, and a colon.

Q: What is the purpose of the return statement?

Q: What are function arguments?

Q: What is the difference between parameters and arguments?
