Q: What is Java and what are its main features?
A: Java is a high-level, object-oriented, and secure programming language. Its most prominent feature is "Write Once, Run Anywhere" (WORA), which means that compiled Java code can run on all platforms that support Java without the need for recompilation, thanks to the JVM (Java Virtual Machine).
